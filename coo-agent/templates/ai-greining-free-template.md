**Generation prompts for each section:**

### Industry Analysis Prompt:
```
Industry: {INDUSTRY}
Pain points: {PAIN_POINTS}

Generate:
1. What are common AI use cases in {INDUSTRY}?
2. What are competitors doing?
3. What is realistic for a team of {TEAM_SIZE}?
Keep it brief (2-3 sentences max).
```

### Opportunity Identification Prompt:
```
Company: {COMPANY_NAME}
Industry: {INDUSTRY}
Pain points: {PAIN_POINTS}
Team size: {TEAM_SIZE}
Main goal: {MAIN_GOAL}

Generate 3-5 AI opportunities ranked by:
- ROI potential
- Implementation difficulty
- Time to value

For each opportunity, provide:
1. Name (short, clear)
2. Description (2-3 sentences)
3. Benefits (3 bullet points)
4. Estimated value (time saved, cost saved, quality improvement)
5. Difficulty level (Easy/Medium/Hard)
6. Priority score (1-5)
7. Next steps (3 high-level steps)

Make it actionable but not too detailed (that's the paid product).
```

### Prioritization Prompt:
```
Given these opportunities: {OPPORTUNITIES}

Recommend:
1. Which to start with and why
2. Which to do second and why
3. What to measure

Keep it simple and clear.
```
